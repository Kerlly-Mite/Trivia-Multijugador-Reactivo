# 🎮 Trivia Multijugador Reactivo

Sistema web de trivia multijugador desarrollado en **Python** utilizando **ReactPy**, **Asyncio** y **MySQL**, permitiendo partidas en tiempo real con múltiples jugadores conectados simultáneamente.

## 🚀 Tecnologías

- Python 3.x
- ReactPy
- Asyncio
- MySQL

## 👥 Integrantes

- Integrante 1 – Frontend ReactPy
- Integrante 2 – Lógica del juego y Asyncio
- Integrante 3 – Base de datos y persistencia

## 🎯 Objetivo

Desarrollar una aplicación web de trivia multijugador con una interfaz reactiva, un estado global compartido, tareas asíncronas concurrentes y persistencia de información mediante una base de datos relacional.
